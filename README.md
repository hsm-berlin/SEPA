## SEPA

Erweiterung für Joomla zur Generierung von SEPA-Lastschriften als XML-Datei.


### Hinweise
Wenn Sie die Klasse nicht mit Joomla verwenden, sollten Sie die Zeile:
```
defined( '_JEXEC' ) or die( 'Restricted access' );
```
löschen.
